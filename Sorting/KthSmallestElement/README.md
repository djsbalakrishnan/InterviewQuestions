## Matrix Search
Find the Bth smallest element in given array A .
NOTE: Users should try to solve it in less than equal to B swaps.

### Constraints
```
1 <= |A| <= 100000
1 <= B <= min(|A|, 500)
1 <= A[i] <= 1e9
```

### Examples
###### Input
```
A = [2, 1, 4, 3, 2]
B = 3
```
###### Output
```
2
```
###### Input
```
A = [1, 2]
B = 2
```
###### Output
```
2
```
