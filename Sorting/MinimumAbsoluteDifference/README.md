## Minimum Absolute Difference
Given an array of integers A, find and return the minimum value of | A [ i ] - A [ j ] | where i != j and |x| denotes the absolute value of x.

### Constraints
```
2 <= length of the array <= 1e6
-1e9 <= A[i] <= 1e9
```

### Examples
###### Input
```
A = [1, 2, 3, 4, 5]
```
###### Output
```
1
```
###### Input
```
A = [5, 17, 100, 11]
```
###### Output
```
6
```
