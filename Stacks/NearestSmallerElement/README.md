## Nearest Smaller Element
Given an array of integers, find the nearest smaller number for every element such that the smaller element is on left side.

### Constraints
```
1 <= |A| <= 1e5
-1e9 <= A[i] <= 1e9
```

### Examples
```
Input: {1, 6, 4, 10, 2, 5}
Output: {-1, 1, 1,  4, 1, 2}
```
```
Input: {1, 3, 0, 2, 5}
Output: {-1 , 1, -1, 0, 2}
```
