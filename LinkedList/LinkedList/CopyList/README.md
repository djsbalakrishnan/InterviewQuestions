## CopyList
A linked list A is given such that each node contains an additional random pointer which could point to any node in the list or NULL.

Return a deep copy of the list.

#### Constraints
`1 <= N <= 1e5`

#### Example Input
`Given list`
   `1 -> 2 -> 3`
`with random pointers going from`
  `1 -> 3`
  `2 -> 1`
  `3 -> 1`

#### Example Output
`Given list`
   `1 -> 2 -> 3`
`with random pointers going from`
  `1 -> 3`
  `2 -> 1`
  `3 -> 1`
