## Reorder Linked List
Given a singly linked list A
`A: A0 → A1 → … → An-1 → An `

reorder it to:
`A0 → An → A1 → An-1 → A2 → An-2 → … `

You must do this in-place without altering the nodes' values.

#### Constraints
`1 <= N <= 1e6`

#### Example Input
`1 -> 2 -> 3 -> 4 -> 5 -> 6`

#### Example Output
`1 -> 6 -> 2 -> 5 -> 3 -> 4`
