## N Integers containing only 1, 2, and 3
```
Given an integer A. Find and Return first positive A integers in ascending order containing only digits 1, 2 and 3.
```

### Constraints
```
1 <= A <= 29500
```

### Examples
```
Input 1: A = 3
Output 1: [1, 2, 3]


Input 2: A = 7
Output 2: [1, 2, 3, 11, 12, 13, 21]
```
