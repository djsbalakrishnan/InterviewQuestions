## Nodes Sum
You are given the root node of a binary tree A. You have to find the sum of node values of this tree.

### Constraints
```
1 <= |A| <= 1e5
0 <= Value of each node <= 1e4 
```

### Examples
###### Input
```
 Values =  1      
          / \     
         8   3                       
        /         
       2                                     
```
###### Output
```
 14 
```
