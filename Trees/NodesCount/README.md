## Nodes Count
You are given the root node of a binary tree A. You have to find the number of nodes in this tree.

### Constraints
```
1 <= |A| <= 1e5
0 <= Value of each node <= 107 
```

### Examples
###### Input
```
 Values =  1      
          / \     
         4   3                       
        /         
       2       
```
###### Output
```
 4
```
