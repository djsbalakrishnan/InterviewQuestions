## Kth Smallest Element In BST
Given a binary search tree represented by root A, write a function to find the Bth smallest element in the tree.

### Constraints
```
1 <= Number of nodes in binary tree <= 1e5
0 <= node values <= 1e9 
```

### Examples
###### Input
```
    2
   /  \
  1    3
B = 2
```
###### Output
```
   2
```

###### Explanation 
```
2nd element is 2.
```
