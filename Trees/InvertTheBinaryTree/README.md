## Invert the Binary Tree
Given a binary tree A, invert the binary tree and return it.
Inverting refers to making left child as the right child and vice versa.

### Constraints
```
1 <= size of tree <= 1e5
```

### Examples
###### Input
```
     1
   /   \
  2     3
 / \   / \
4   5 6   7
```
###### Output
```
     1
   /   \
  3     2
 / \   / \
7   6 5   4
```
###### Explanation
```
Tree has been inverted.
```
